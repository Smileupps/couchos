Smileupps Couchapp OS

A web interface for your Couchapps

  • This app provides a Couchapp Editor, a Bulk Docs Editor and restricted admin-only access to Futon and Fauxton
  • This app is a couchapp itself, served from your CouchDB instance, stored in /ceditor/_design/ceditor. Please don't remove it!!!
  • Source code of this app can be found on Github
